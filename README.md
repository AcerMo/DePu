# Texas Hold'em Online (多人在线德州扑克移动端小游戏)

这是一个专为手机端访问设计的、支持多人在线联机同时游玩的德州扑克小游戏。

项目采用 **Python (FastAPI + WebSockets) 作为后端** 处理游戏逻辑并维护同步状态，使用 **原生 HTML5 / CSS3 (深色玻璃化微光视觉系统) / Vanilla JS 作为前端** 提供丝滑的游戏动画与响应式操作。

---

## 核心特性
1. **轻量即插即用**：前后端整合在同一端口（通过 FastAPI 服务托管静态资源），本地一键轻松运行。
2. **手机多端适配**：响应式竖屏设计，大按钮与滑动输入条，非常适合移动端浏览器与微信内置浏览器单手把玩。
3. **保密与防作弊**：所有核心发牌、状态计算皆在后端完成。每个客户端连接仅接收其个人的手牌信息，在摊牌阶段前绝不泄露他人底牌。
4. **全自动边池（Side Pots）**：内置科学的筹码边池拆分与结算算法，支持多人不同额度 All-in 时的精确配发。
5. **社交日志系统**：集成局内操作日志与实时聊天室。

## 本地运行说明

我们推荐使用 Python 3.12 直接启动本程序。

### 1. 安装依赖

打开终端（PowerShell 或 CMD），进入项目根目录运行：
```bash
pip install -r requirements.txt
```

### 2. 启动服务

运行以下命令开启 FastAPI 服务器：
```bash
python -m uvicorn backend.server:app --host 0.0.0.0 --port 8000 --reload
```

### 3. 多人访问游玩
- **电脑访问**：在浏览器中打开 `http://localhost:8000`。
- **手机访问（同局域网内）**：
  1. 获取你电脑的局域网 IP（例如在 Windows 终端中输入 `ipconfig`，找到 IPv4 地址为 `192.168.x.x`）。
  2. 在同一局域网下的手机浏览器（或微信、Safari）中打开 `http://192.168.x.x:8000`。
  3. 输入各自的玩家昵称，输入或保持相同的**房间号**（例如默认 `1001`），点击**进入房间**即可开始联机对局！

## 游戏核心规则与座位使用说明
- **座位**：进入房间后默认为“旁观者”。点击桌面上虚线框的**“坐下”**按钮选择一个座位入局，初始筹码为 `$10000`。
- **开始游戏**：当有 **2 名或以上** 有筹码的玩家坐下后，点击下方的**“开始游戏”**按钮即可开局。
- **操作规则**：
  - **弃牌 (Fold)**：放弃本局。
  - **过牌 (Check)**：在当前无需跟注时过牌。
  - **跟注 (Call)**：跟上当前最大下注额。若筹码不足自动转化为 All-in。
  - **加注 (Raise)**：增加当前投注额，通过操作下方的滑杆拖拽，或者快捷按钮选择（如最小加注、2倍/3倍大盲、底池大小加注或 All-in）。
- **重买/补充筹码**：如果输光了筹码，在当前牌局结束后，可以点击下方的**“补充筹码”**一键恢复至 `$10000` 继续游戏。

## 线上部署指南 (Render 一键部署)

由于本系统将前端打包托管在 Python 后端下，在支持 Python/Docker 托管的平台（推荐 Render）中部署极为省心：

1. **新建 GitHub 仓库**，将此 WEBUI 下的所有内容推送上去。
2. 登录 [Render.com](https://render.com/)，在 Dashboard 中点击 **"New" -> "Web Service"**。
3. 连接你的 GitHub 仓库。
4. 在配置页面中：
   - **Runtime**: `Python`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python -m uvicorn backend.server:app --host 0.0.0.0 --port $PORT`
5. 点击 **Deploy**，部署完成后平台会提供一个专用的 `https://xxx.onrender.com` 链接，任何人点开此链接即可通过手机在线联机游玩！
